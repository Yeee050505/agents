执行流程（一次正常对话）:
─────────────────────────────────────────────────
1. POST /api/v1/chat {"message":"写个快排"}
2. intent_node → LLM cache命中 → intent="code"
3. code_agent → LLM cache命中 → 返回快排代码
4. supervisor → 检查通过 → END
5. 总耗时: 0.0s (全缓存), 8.5s (首次LLM), 30s (DeepSeek)
